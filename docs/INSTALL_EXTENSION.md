# Browser Extension Installation Guide

This is a developer-mode Chrome / Edge extension.

It is not a Chrome Web Store release. It must be loaded manually.

## Quick Steps

1. On this repository page, click **Code**.
2. Click **Download ZIP**.
3. Unzip the downloaded file.
4. Open `chrome://extensions` for Chrome or `edge://extensions` for Edge.
5. Turn on **Developer mode**.
6. Click **Load unpacked**.
7. Select the `extension` folder inside the unzipped repository.
8. Open **Artificial Wisdom Guardrail Prompt** from the extension icon.
9. Click **Copy preview content**.
10. Paste it into an AI only if you choose.

## Safety Notes

This extension does not read page content.

- It does not analyze the current web page.
- It does not send content to external AI.
- It does not use any AI API.
- It does not capture page text, sidebars, ads, or menus.

## When to Use It

Most users only need the `index.html` copy page.

This extension is mainly for testers who repeatedly use the Artificial Wisdom Guardrail Prompt across multiple AI systems.
