# ブラウザ拡張のインストール方法

この拡張機能は Chrome / Edge 用の開発版です。

Chrome Web Store から入れる完成版ではありません。手動で読み込む必要があります。

## 一番簡単な流れ

1. リポジトリ画面で **Code** を押す。
2. **Download ZIP** を押す。
3. ZIPを解凍する。
4. Chromeなら `chrome://extensions`、Edgeなら `edge://extensions` を開く。
5. **デベロッパーモード** をオンにする。
6. **パッケージ化されていない拡張機能を読み込む** を押す。
7. 解凍したフォルダ内の `extension` フォルダを選ぶ。
8. 拡張機能アイコンから **Artificial Wisdom Guardrail Prompt** を開く。
9. **プレビュー内容をコピー** を押す。
10. 任意でAIに貼り付ける。

## 注意

この拡張機能は、ページ本文を読みません。

- 現在のWebページを解析しません。
- 外部AIへ自動送信しません。
- AI APIを使いません。
- ページ内容、左メニュー、広告、サイドバーを取得しません。

## 使う意味

多くの人は、`index.html` のコピー用ページだけで十分です。

この拡張機能は、複数のAIで何度も人工叡智ガードレールプロンプトを使う検証者向けです。
