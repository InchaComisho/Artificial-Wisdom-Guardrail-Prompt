# Artificial Wisdom Guardrail Prompt Extension

This is a prompt-only Chrome / Edge developer extension.

It does not read page content and does not use any AI API.

## Install

1. Download this repository via **Code → Download ZIP**.
2. Unzip it.
3. Open `chrome://extensions` or `edge://extensions`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select this `extension` folder.

## Use

Open the extension and click **Copy preview content**.

Paste the prompt into an AI only if you choose.
