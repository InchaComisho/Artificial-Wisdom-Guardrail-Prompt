# Artificial Wisdom Guardrail Prompt

[日本語版はこちら](README_ja.md)

A simple, install-free copy page for the **Artificial Wisdom Guardrail Prompt**.

Public prompt page: <https://inchacomisho.github.io/Artificial-Wisdom-Guardrail-Prompt/>

This repository is the lightweight user-facing entry point for applying the Artificial Wisdom Guardrail before using AI tools, code agents, automation, decision-support systems, or design proposals.

## Use in 10 Seconds

1. Open the prompt page.
2. Press **Copy Prompt**.
3. Paste it into an AI before your task.
4. Optionally report test results in GitHub Discussions.

## What This Is

This is not an AI system.

It is a prompt distribution page and optional browser extension source package.

It does not:

- read the current page,
- send page content to external AI,
- use an AI API,
- install anything automatically,
- claim certification, scientific proof, or completed implementation.

## What the Prompt Does

The prompt asks an AI to evaluate a task, proposal, output, code change, or automation through the following Six Principles:

1. Natural Law
2. Harmony
3. Circulation
4. Structure
5. Order
6. Wa

It also asks the AI to avoid optimizing only for speed, scale, automation, profit, capability, or short-term success.

## Recommended Use

Use this prompt before asking an AI to:

- generate code,
- automate a process,
- evaluate a design proposal,
- review a climate or ecological claim,
- summarize a high-impact idea,
- make decisions that affect people, society, nature, or long-term systems.

## Browser Extension

The browser extension is optional. Most users can simply use the copy page.

The extension is mainly for people who want to repeatedly test the guardrail prompt across multiple AI systems.

To use it, download the repository via **Code → Download ZIP**, unzip it, and load the `extension` folder in Chrome / Edge developer mode.
