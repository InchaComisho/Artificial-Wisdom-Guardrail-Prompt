# 人工叡智ガードレールプロンプト

インストール不要で使える、**人工叡智ガードレールプロンプト** のコピー用ページです。

このリポジトリは、AIツール、コードエージェント、自動化、意思決定支援、設計提案を使う前に、人工叡智ガードレールを適用するための軽量な入口です。

## 10秒で使う

1. コピー用ページを開く。
2. **プロンプトをコピー** を押す。
3. AIに貼り付けてからタスクを依頼する。
4. 必要なら GitHub Discussions にテスト結果を報告する。

## これは何か

これはAIではありません。

プロンプト配布ページと、任意のブラウザ拡張ソースです。

このページは以下を行いません。

- 現在ページの本文を読む
- ページ内容を外部AIに送る
- AI APIを使う
- 何かを自動インストールする
- 科学的証明・認証・完成済み実装を主張する

## プロンプトの役割

このプロンプトは、AIに対して、タスク・提案・出力・コード変更・自動化を次の六つの理で評価するよう求めます。

1. 摂理
2. 調和
3. 循環
4. 構造
5. 秩序
6. 和

また、速度、規模、自動化、利益、能力、短期的成功だけを最適化しないよう求めます。

## 推奨用途

次のような依頼をAIにする前に使います。

- コード生成
- 作業の自動化
- 設計提案の評価
- 気候・生態系に関わる主張の確認
- 影響の大きい思想・提案の要約
- 人間、社会、自然、長期システムに影響する判断

## ブラウザ拡張

ブラウザ拡張は任意です。ほとんどの人はコピー用ページだけで十分です。

拡張機能は、複数AIで何度もガードレールプロンプトを試したい検証者向けです。
